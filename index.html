What is an LLM?

A Large Language Model (LLM) is an AI system trained on massive text datasets. It can understand prompts, answer questions, generate text, summarize content, and even write code.

What is a Token?

Tokens are small pieces of text processed by an AI model. Pricing and context windows are usually measured using tokens.

How an LLM Works

  1. 1

    User Input

    The user writes a prompt into the interface.

  2. 2

    Tokenisation

    The prompt is split into smaller text tokens.

  3. 3

    Embedding

    Tokens are converted into vector representations.

  4. 4

    Attention Layers

    The model analyzes relationships between tokens.

  5. 5

    Prediction

    The AI predicts the next most likely token.

  6. 6

    Output Generation

    The generated tokens become the final response.
